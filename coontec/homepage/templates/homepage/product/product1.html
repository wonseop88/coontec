개요

Imperas 솔루션은 System Full Virtualization(전가상화) Simulator 입니다. 임베디드 시스템의 CPU 뿐 아니라 Peripheral Device, Bus, Memory 등을 모두 가상화하여 실제 Target에서 실행 되는 바이너리를 개발 환경에서 그대로 수행할 수 있도록 하며 CPU, 디버깅, 데이터 가상입력등을 수행하여 다양한 분야에 활용할 수 있습니다.

특징 및 장점

임베디드 시스템 보다 수십~수백만배 이상 빠른 수행 속도 가상시스템을 활용한 다양한 기능 제공 (디버깅, 테스팅, 분석, 오류 주입, 커버리지, OS Tracing, Cash정보 등) ARM, MIPS, PowerPC, Synopsys, Renesas, openCores, Altera 등 지원 자체 개발된 CPU Core 개발을 위한 Tool 제공 Eclipse, GDB, 디바이스 생성기 등 제공 OVP Projects 를 기반으로한 디바이스, CPU Core 지원 생태계 제공

주요기능

분류 VAP주요기능 설명
Tracing Mark Processor(Event, Register 등) Architecture 내부 데이터 event 추적 기능
Function Tracing Call 추적 및 참조 횟수, 함수 데이터 추적 기능
OS Task Tracing 다양한 RTOS 에 대한 Event, task 추적 기능
Coverage Code Coverage Code 수행 및 조회에 대한 커버리지 데이터 제공
Instruction Coverage Instruction 단위의 실제 커버리지 데이터 제공
Analysis Memory Analysis 메모리 에러 유형 데이터 제공 (Access, Bound, Usage 등)
Cache Analysis Cache 프로파일 기능 제공 (hit miss, context 등)
Exception & Interrupt Analysis Runtime error 에 대한 분석 데이터 제공
OS Scheduler Analysis OS Task 에 대한 Schedule profile 제공
Testing Fault Injection Fault 데이터 자동 생성 주입으로 오류 검출
Protocol Verification 프로토콜 데이터에 대한 검증 환경 지운
Shared Resource Introspection 공유 자원에 대한 테스트 기능 제공 (메모리 등)
